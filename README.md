## Command
### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `yarn test`

Launches the test runner in the interactive watch mode.

## About Project

### Project structure
    - assets
        neccessary images
    - components
        reusable components
    - pages
        page components
    - redux
        define store, reducer and saga
    - services
        define global functions, axios functions and constants.
    - App.scss
        define styles
    - App.test.js
        unit test code here.
